# 芙蓉绽

（1~41待补档）

## 42

外面的天色已然变暗，屋内也黑蒙蒙一片，敖寸心走进房内时差点被绊倒，她刚要发脾气看见了直挺挺躺在床上一言不发只是胸口时不时起伏的杨戬便又将话吞进肚中去，摸索着将蜡烛点亮。烛火一跳一跳的，火光映在杨戬脸上更衬得他面色阴晴不定，和雷雨前的天空一个颜色。
<!-- more -->
敖寸心深吸了一口气合衣躺在了杨戬身边，杨戬并不理她，他的脸上带着种说不清道不明的冷淡，龙女细声细气地唤了他一声他也不吱声，勾了勾他手指杨戬也没反应，敖寸心几次三番动作都没回应她也急了，直接扯掉了杨戬的腰带拉下他裤子就要把那话儿往嘴里含。

杨戬的阳物对她并不大的嘴巴算庞然巨物，俨然快霸占了全部空间，敖寸心有些吃力但是并不准备松口，她耐心地舔舐着这话儿，可能是太过于熟悉杨戬的阳具不一会儿就完全勃起顶在了龙女的喉咙上，又硬又灼热让敖寸心有些害怕想松口，在她舔舐的时候没注意到身上人目光一直没有移开，只是相比于往日的浓情蜜意显得有些阴鸷。松开后这人阳具的气息便扑鼻而来，不重但很凛冽，隐隐带着些皮革和松木香味，紫红一根全然硬起，龟头大如鸽卵顶端带着些许蜜露，她看得有些馋自个儿解开了裙衫亵裤要坐下去，却被杨戬躲开扑了个空。白嫩嫩一对膝盖正好摔到了床榻上，不算很疼但也略难受。龙女心内一半是气愤一半是急色，她的手捏着杨戬的命根指尖摩挲着穴眼，另一只手轻轻地揉弄着自己的乳肉，将这一团白腻细滑摆弄给他看，美色当前老君也得多念几卷《清心咒》，可是杨戬仍旧无动于衷面上无悲无喜，似乎是个坐化的佛像。良久待这人阳物已经烫如岩浆一般他仍没动作，任敖寸心口干舌燥他衣物都没少一件，搞得龙女无可奈何寻思着杨戬怎么妹妹来了一趟就转性了，眼看久久攻不下城她也没兴致了，收了手敛敛衣襟准备睡个素觉。不料她刚躺下杨戬便趴到了她身上，两条精壮的臂膀撑在了她肩旁，他口中呼出的热气正对着她的脖颈，敖寸心心内不由得一紧，她借着烛火打量着杨戬的脸色，只是越看心里越没底。

杨戬心内是极烦躁的，睡醒后梦中事依然在脑中挥之不去——睡梦虽假但梦里眼见得寸心和那二郎洞房花烛同饮交杯酒还要当着他的面行房时他恨不得自己早死在了华山的心是千真万确的。若是当日寸心没来寻他，或是轻易就被他赶走了，久而久之会不会忘记三界还有他这么个人？会不会与他人再结鸳盟？他越想越后怕越想越心惊，龙女白嫩的乳肉和细腻的掌心撩拨得他心绪不宁恨不得提枪上阵听她娇喘吁吁，可是他还是忍住没动作，只因看着这女郎费尽心思讨好他他心里才好受些。他知道不该迁怒于她那毕竟是他擅自做梦，可当她主动撩拨的身影与梦中重合时他的心仿佛也在油锅上煎烤，杨戬抑制不住自己，抑制不住想要撕碎她蹂躏她的欲望。

敖寸心一向是个三心二意的主，看他没动作居然准备偷懒赖下。瞥见她穿好了衣服躺在他身边杨戬心中无名火烧得更厉害了，把他撩拨得兴起自己却准备睡去，莫非留着力气给那个二郎吗？

敖寸心心中正慌乱衣襟却被粗暴地扯开，她的乳肉毫无遮掩的暴露出来，杨戬此刻的眼神充满了欲望和渴求，看起来像极了发情的野兽，烛光下更衬得眼中带些阴鸷，龙女怕极了想逃，可是能逃哪里去呢，她甚至都逃不出卧室，更遑论灌江口杨府。敖寸心眼泪都快出来了仍咬咬嘴唇努力不让泪珠子掉下来“杨戬你别这样，我们是夫妻”，杨戬并不搭话直接咬上了她的乳肉，他力道极大又咬又吸，似乎是恨不得把龙女吞下腹中，手也丝毫不安分，直接探入敖寸心花穴中抠弄。这下龙女可受了大罪，胸前两团脂膏一样的白腻被放肆地咬着，乳晕和乳蒂也被用力嗦着，杨戬的力道并不像是调情倒像是刑罚，花穴的嫩肉也被肆意抠弄着，那人的指节时不时擦过她娇嫩的肉壁，闹得她又疼又爽快，里面积攒的春水也迫不及待地喷出来迎合着。敖寸心只觉得一会儿被烈焰炙烤着一会儿被寒冰冻着，哭得声音都哑了“我又没对不住你干嘛要弄疼我”，杨戬咬得她双乳遍是齿痕后才松开牙齿，他居高临下地看着妻子，眼中情欲更盛“未雨绸缪，免得你再生了要跑的心思”。敖寸心听了这话心中只有无限的凄惶，她抽噎地看着杨戬“我什么时候生了要跑的心思，我对你从来都是一心一意的”，锁骨却被啮咬着，杨戬的话有些含糊不清“那就让我验验”。

杨戬上天后跟以前相比变了许多，世人都道他奸佞权臣，敖寸心并不相信可是杨戬性子还是跟以前不大一样。从前在杨府她说什么是什么，这人每每吵不过她总是气得面红耳赤夺门而去，上了一次天后杨戬不但会跟她调情逗乐，唇舌功夫也厉害许多，只是心思变得比往日更难猜。今日他说出这话龙女知道她只能应着。
